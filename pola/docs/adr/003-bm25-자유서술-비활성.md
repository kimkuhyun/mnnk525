# ADR 003 — 자유서술 카테고리에서 BM25/Rerank 비활성

**상태**: 적용 · 2026-05-25

## 결정
`CATEGORY_BM25_DISABLE = {"자유서술"}`, `CATEGORY_RERANK_DISABLE = {"자유서술", "비교"}`.

## 배경
- BM25 활성: 자유서술 0.633 → 0.533 (역효과)
- Rerank 활성: 자유서술 변화 없음 (쿼리 짧음의 한계)

## 진단
- 자유서술 쿼리 = "회사명 + 섹션 헤더" (10자 내외, 의미 검색이 키워드 매칭보다 효과적)
- BM25 가 본문 키워드 매칭하면 보일러플레이트가 top 점령
- Cross-encoder rerank 도 짧은 쿼리에 약함

## 결과
자유서술은 dense top-K + Contextual prefix 단독으로 0.867 달성.

다른 카테고리 (정형/시계열/출처) 는 BM25 + Rerank 활성으로 0.86~1.0.
