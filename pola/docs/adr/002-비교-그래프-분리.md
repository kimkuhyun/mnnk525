# ADR 002 — 비교 카테고리 = Neo4j Cypher 영역

**상태**: 적용 · 2026-05-25

## 결정
비교 카테고리 ("A vs B 2024년 부채총계") 평가를 벡터 게이트에서 분리, 그래프 게이트(`polaris graph-eval`) 로 별도 측정.

## 배경
벡터 검색으로 비교 카테고리 recall@10 0.29~0.64 한계. 자연키 확장된 expected ~16개 / top-10 → 천장 0.7.

## 진단
비교 쿼리는 본질적으로 **두 정형 값 비교**. 설계 4-Tier 격리에서 정형 사실은 **Tier 1 (Neo4j)** 영역.

## 결정 근거
Cypher 한 줄로 100% 정확:
```cypher
MATCH (o:Organization)-[:HAS_METRIC]->(m:FinMetric)
WHERE o.corp_code IN $corps AND m.year=$year AND m.indicator=$indicator
RETURN DISTINCT o.corp_code
```

## 결과
- 그래프 F1 = **1.000**, 30/30 exact match
- 벡터 게이트는 6/6 PASS (비교 제외)

## 추가 작업
설계상 FinMetric 노드가 정의됐지만 적재 미완 → `scripts/admin/load_finmetric.py` 추가 (18,077 노드 적재).
